## 4. include.js （共通読み込みスクリプト）

```js
// js/include.js
document.addEventListener("DOMContentLoaded", () => {
  // ヘッダー読み込み
  fetch("header.html")
    .then(response => response.text())
    .then(data => {
      document.getElementById("header").innerHTML = data;
    });

  // フッター読み込み
  fetch("footer.html")
    .then(response => response.text())
    .then(data => {
      document.getElementById("footer").innerHTML = data;
    });
});
```
